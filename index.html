Data Structures in C

This is a collection of six projects that I developed in 2019 for my Data Structures class at University. All of the projects were implemented using the C language only.
1. I coded a stack of strings using an array. Push() and Pop() operations are included.
2. This program allows you to store numbers in either a Stack or a Queue. The user may also change the data structure even after storing initial values.
3. This program implements a Graph to store geographical information about the Mexican Republic.
4. The purpose of this activity was to implement a recursive quick sort algorithm to sort 20 song titles in ascending and descending order.
5. The goal for this project was to implement a linear search algorithm allowing the user to store an indefinite amount of numbers and then search for any given value.
6. The goal for the last project was to implement a binary search tree with data provided by the user. The program:
a) Accepts a new number and adds it accordingly to the binary tree.
b) Performs binary search on a given value.
c) Prints the binary tree.
